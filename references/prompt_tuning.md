# 转写调优指南 (Prompt Tuning)

> 各语言 `initial_prompt` 调优策略，基于 Whisper 论文和实践。v0.1.0

## 核心原理

Whisper 的 `initial_prompt` 提供上下文提示，帮助模型消除歧义：

- 提供该领域**高频专有名词**（人名/公司名/术语）→ 降低误识率
- 提供该语言的**典型句式** → 引导模型正确的语言模型分布
- 不超 200 字符（过长无增益，浪费 token）

## 按语言分类的 initial_prompt 模板

### 日语 (ja)

```
これは稲盛和夫氏の京セラフィロソフィについての講話です。
```

**适用场景**: 稻盛哲学/京瓷/经营哲学讲座

**通用日语模板**:

```
これはビジネスに関する日本語の講演です。経営、哲学、人生について話しています。
```

**调优要点**:

- 包含主讲人姓名（稲盛和夫、松下幸之助...）
- 包含主题词（経営、哲学、人生、働き方）
- 日语敬语和谦让语识别：提示句中含「です・ます」体可改善正式场合的识别

### 中文 (zh)

```
这是一场关于企业经营管理哲学的中文演讲。
```

**适用场景**: 商业讲座/培训/会议

**通用中文模板**:

```
这是一场关于[主题]的中文演讲。主要内容包括：[关键词1]、[关键词2]、[关键词3]。
```

**调优要点**:

- 中文同音词极多，`initial_prompt` 中的领域词能大幅降低同音歧义
- 如果讲话者带方言口音：在 prompt 中加「带有[XX]口音」（对模型有一定引导效果）
- 中英夹杂场景：prompt 中混入对应的英文术语

### 英语 (en)

```
This is a business lecture on management philosophy and corporate culture.
```

**调优要点**: 英语模型训练数据最丰富，`initial_prompt` 对常规语音影响较小。重点在：

- 专有名词（公司名/品牌名/人名）
- 缩略语（KPI, OKR, ROI...）

### 多语言混合

```
language=None  # 让模型自动检测
initial_prompt=""  # 空提示让模型自由发挥
```

混合语言场景：

- 保持 `language=None`（自动检测）
- 如果已知主语言，设置 `language` 为最常出现的语言
- 在 initial_prompt 中用该语言的常见词作引导

## VAD 参数调优

| 场景                | min_silence_duration_ms | speech_pad_ms | 效果           |
| ------------------- | ----------------------- | ------------- | -------------- |
| 讲座/演讲（长停顿） | 700-1000                | 500           | 减少段落碎片化 |
| 会议/对话（短停顿） | 300-500                 | 300           | 保留对话节奏   |
| 采访（问答交替）    | 400-600                 | 400           | 平衡           |
| 默认                | 500                     | 400           | 通用           |
