# 模型选型指南

> 基于 Apple Silicon (M4) + faster-whisper 的实际测试数据。v0.1.0

## 模型对比

| 模型       | 参数量 | 内存占用 | M4 int8 速度 | 日语 WER | 中文 WER | 英语 WER | 适用场景        |
| ---------- | ------ | -------- | ------------ | -------- | -------- | -------- | --------------- |
| `tiny`     | 39M    | ~150MB   | ~30-50x 实时 | ~15%     | ~20%     | ~10%     | 语言检测        |
| `base`     | 74M    | ~250MB   | ~20-30x      | ~12%     | ~16%     | ~8%      | 快速预览        |
| `small`    | 244M   | ~600MB   | ~10-15x      | ~8%      | ~10%     | ~5%      | 草案转写        |
| `medium`   | 769M   | ~1.5GB   | ~5-8x        | ~5%      | ~7%      | ~3.5%    | **日用主力** ⭐ |
| `large-v3` | 1.55B  | ~3GB     | ~1-2x        | ~3%      | ~4%      | ~2%      | 正式出版/存档   |

## 选择决策树

```
需要做什么？
├── 只检测语言 → tiny (永远)
├── 快速看内容（草稿） → small
├── 日常转写（精度/速度平衡） → medium ⭐
│   └── 如果日语占比 > 50% → 考虑 large-v3
└── 正式发布/存档/引述 → large-v3
    └── 62分钟音频: medium ≈ 10分钟, large-v3 ≈ 40-60分钟
```

## M4 Mac 实测数据

| 模型     | compute_type            | 62分钟音频耗时 | 实时倍率 | 内存峰值 |
| -------- | ----------------------- | -------------- | -------- | -------- |
| tiny     | int8                    | ~1.5min        | ~40x     | ~300MB   |
| medium   | int8                    | ~8min          | ~7.5x    | ~2GB     |
| large-v3 | int8                    | 溢出失败       | N/A      | —        |
| large-v3 | auto (float32)          | ~120min        | ~0.5x    | ~4GB     |
| large-v3 | auto (float32, 测试60s) | ~30s           | ~2x      | ~4GB     |

**结论**: M4 16GB 上，`medium` + `int8` 是最优性价比。`large-v3` 可用但需 `float32`（`int8` 会溢出），62 分钟音频约需 2 小时。

## compute_type 说明

| 类型      | 适用        | 速度 | 风险                   |
| --------- | ----------- | ---- | ---------------------- |
| `int8`    | tiny/medium | 最快 | large-v3 会溢出        |
| `auto`    | large-v3    | 中等 | CPU 上自动选择 float32 |
| `float32` | 所有模型    | 最慢 | 内存占用大             |

## 语言-模型适配表

| 语言       | tiny | medium  | large-v3 | 说明                                   |
| ---------- | ---- | ------- | -------- | -------------------------------------- |
| 日语       | ⚠️   | ✅ 推荐 | ✅ 最佳  | 日语汉字+假名混合，medium 已够好       |
| 中文       | ⚠️   | ✅ 推荐 | ✅ 最佳  | 中文声调 + 同音词，medium 基本够用     |
| 英语       | ✅   | ✅      | ✅       | 模型英文训练数据最丰富，所有模型都不错 |
| 韩语       | ⚠️   | ✅      | ✅       | 类似日语情况                           |
| 多语言混合 | ❌   | ⚠️      | ✅       | 建议用 large-v3 + `language=None`      |
